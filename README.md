# Основы Python
Этот репозиторий содержит конструкции кода на Python, которые были написаны мною и оказались полезными при изучении различных типов данных, условий и циклов, функций, ООП и классов в данном языке программирования.

Название файлов соответствуют реализованным задачам.

## Использование
Быстрый доступ к готовым блокам кода из данного раздела помогает при написании ёмких проектов, ориентируясь на практику ранее написанных скриптов.
